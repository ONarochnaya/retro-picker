# retro-picker

A small React app build with Vite for rebuy Hackathon 2025. The goal is to help teams quickly pick a **retrospective format**.

## Quick start

### Install & run

```bash
git clone https://github.com/ONarochnaya/retro-picker.git
cd retro-picker
npm install

# start dev server
npm run dev

# build for production
npm run build
```

## Tech stack

- React + Vite
- ESLint
